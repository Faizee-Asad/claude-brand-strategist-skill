# Claude Brand Strategist Skill

A Claude Code skill for brand strategy, brand identity, positioning, messaging, tone of voice, visual identity direction, social media branding, and brand guidelines.

## Author

Created by **Asad Faizee**.

Replace this with your real name, brand name, or GitHub username.

## Overview

`claude-brand-strategist-skill` helps Claude produce structured branding work for businesses, startups, creators, products, campaigns, and personal brands.

It can help with:

- Brand strategy
- Brand positioning
- Brand identity direction
- Brand messaging
- Tone of voice
- Taglines and slogans
- Audience personas
- Competitor positioning
- Visual identity guidance
- Social media branding
- Brand audits
- Brand guidelines

## Installation

Copy the `brand-strategist` folder into your Claude Code skills directory:

```bash
mkdir -p ~/.claude/skills
cp -r brand-strategist ~/.claude/skills/
```

Final path:

```text
~/.claude/skills/brand-strategist/SKILL.md
```
